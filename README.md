# Personal Portfolio

A modern, responsive portfolio website built with React and TailwindCSS to showcase my skills, projects, and experience as a developer.

## 🌐 Live Demo

Visit the live site: [https://unclearyan.github.io/portfolio/](https://unclearyan.github.io/portfolio/)

## ✨ Features

- **Responsive Design** - Fully responsive layout that works seamlessly across all devices
- **Dark/Light Theme Toggle** - Switch between dark and light modes with default dark theme
- **Interactive Sections** - Five main sections providing comprehensive information:
  - **Home** - Landing page with introduction
  - **About** - Personal background with CV view and download options
  - **Skills** - Categorized skill set with proficiency percentages
  - **Projects** - Showcase of completed projects
  - **Contact** - Functional contact form powered by EmailJS
- **CV Integration** - View CV in browser or download directly from the About section
- **Skill Categorization** - Skills organized into Core Languages, Frameworks, and Tools with visual proficiency indicators

## 🛠️ Built With

- **React** - JavaScript library for building user interfaces
- **TailwindCSS** - Utility-first CSS framework for styling
- **EmailJS** - Email service integration for contact form functionality

## 📋 Sections Overview

### Home
Introduction and welcome section to greet visitors.

### About
Personal background and professional summary with integrated CV access:
- View CV in browser
- Download CV option

### Skills
Comprehensive skill set display featuring:
- **Core Languages** - Programming languages with proficiency levels
- **Frameworks** - Framework expertise with percentage indicators
- **Tools** - Development tools and technologies

Each skill includes a visual percentage indicator showing proficiency level.

### Projects
Portfolio of completed projects demonstrating practical application of skills.

### Contact
Functional contact form integrated with EmailJS for direct communication.

## 🎨 Theme

The portfolio features a theme toggle allowing users to switch between:
- Dark Mode (default)
- Light Mode

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
```bash
git clone https://github.com/unclearyan/portfolio.git
```

2. Navigate to the project directory
```bash
cd portfolio
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

5. Open [http://localhost:3000](http://localhost:3000) to view it in your browser
